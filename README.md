## What is the problem solving: 
- Accurate price prediction helps buyers, sellers, and real estate agencies.
- Supports better decision-making and market analysis.

## Steps Done:
- a pipeline to predict house sale prices in King County using machine learning models.

## Dataset Overview:
This dataset contains house sale prices for King County, which includes Seattle. It includes homes sold between May 2014 and May 2015.

- __Size:__ 21,613 rows; 20 features
- __Target:__ price
- __Key features:__ bedrooms, bathrooms, sqft_living, waterfront, view, grade, zipcode, etc.
